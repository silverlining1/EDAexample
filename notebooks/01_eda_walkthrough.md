# EDA Walkthrough

This directory contains Jupyter notebooks for interactive exploration.

## Running Notebooks

```bash
pip install jupyter
jupyter lab
```

## Notebook Index

| Notebook | Purpose |
|---|---|
| `01_eda_walkthrough.ipynb` | Step-by-step EDA on the sample Facebook dataset |
| `02_custom_dataset.ipynb` | Template for running EDA on your own CSV |

## Quick Start

The `main.py` CLI runs the full pipeline non-interactively:

```bash
python main.py --file data/sample_facebook.csv
```

For interactive exploration, open the notebooks and run cells step by step.
